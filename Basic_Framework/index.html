Jedes Element im Periodensystem hat eine spezifische Elektronenkonfiguration – eine bestimmte Anordnung der Elektronen auf den Schalen. Diese Anordnung folgt bestimmten Gesetzmässigkeiten – mit dieser kleinen Anwendungen kannst du diese (hoffentlich) nachvollziehen.

Wenn wir im Periodensystem von Element zu Element gehen, kommt jeweils ein neues Elektron hinzu. Dieses besetzt stets das tiefstmögliche Energieniveau, also die energetisch niedrigste Unterschale. Jede Unterschale hat Platz für eine bestimmte Anzahl Elektronen: s-Unterschalen für 2, p-Unterschalen für 6, d-Unterschalen für 10 und f-Unterschalen für 14 Elektronen. Ist eine Unterschale voll besetzt, geht das nächste Elektron in die nächst höhere Unterschale.
